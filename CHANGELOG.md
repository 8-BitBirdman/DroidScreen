## 🎉 DroidScreen 1.5.1

*To receive a notification on new releases, click on **Watch > Releases only** on the top.*

---

DroidScreen is a macOS-only GUI for [scrcpy](https://github.com/Genymobile/scrcpy).

### Build from source

```shell
git clone https://github.com/8-BitBirdman/droidscreen
cd droidscreen
npm install
npm run build
```

Packaged files are generated to the `build` folder.

Feel free to open issues or PRs for any problems you encounter. Contributions are welcome!
